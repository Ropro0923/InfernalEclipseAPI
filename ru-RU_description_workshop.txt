[h1]Infernal Eclipse of Ragnarok[/h1]
[h3] Ультра-модифицированный аддон для Infernum Mode[/h3]

[b]Ознакомьтесь с соответствующим модпаком здесь:[/b]
https://steamcommunity.com/sharedfiles/filedetails/?id=3456508757

«Infernal Eclipse of Ragnarok» — это ультра-модифицированный аддон для режима сложности Инфернум, который в настоящее время включает в себя расширенную поддержку следующих модов из ассоциированного модпака:
[list]
	[*] Calamity Mod (разумеется)
	[*] Thorium Mod
	[*] Secrets of the Shadows
	[*] Consolaria
[/list]

Для работы этого API необходимы только следующие моды: Calamity Mod, Infernum Mode, Revengence+, Thrower Unification, Infernum Sanctuary WorldGen Adjuster и все их зависимости.

В данный момент этот API добавляет следующее:
[list]
	[*] Более 500 настраиваемых изменений баланса для множества оружия и предметов из включённых модов, чтобы улучшить целостность игрового опыта.
	[*] Объединённые древа крафтов для многих аксессуаров.
	[*] Кросс-модовые интро Инфернума для тех боссов, у которых их ещё не было.
	[*] Корректировка характеристик боссов под режим Инфернум для поддержания постоянной сложности.
	[*] Поддержка босс-раша для всех включённых боссов, а также индивидуальный «6 ранг».
	[*] Новая музыка для определённых этапов игры.
	[*] Изменения в генерации мира для предотвращения конфликтов между модами.
	[*] Исправления ошибок для улучшения совместимости режима Инфернум с мультиплеером.
	[*] Незначительные кросс-модовые улучшения для модов вне основного модпака.
[/list]

Также он включает в себя некоторые встроенные системы:
[list]
	[*] Новую систему объединения классов разбойника/метателя с уникальными скрытными атаками для кросс-модового метательного оружия с использованием [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3535746309] Thrower Unification[/url].
	[*] Новую гибко настраиваемую систему возрождения в мультиплеере, позволяющую установить количество жизней у каждого игрока во время битвы с боссом (по умолчанию 1 жизнь в режиме Инфернум).
	[*] Систему, позволяющую зафиксировать выбранную сложность Calamity.
	[*] Систему, позволяющую призвать Астрагельдона, даже если Лунный лорд уже побеждён.
	[*] Систему, позволяющую играть в режиме Инфернум в мирах со сложностью Мастер и Легенда.
[/list]

В будущем это станет контент-модом с собственной историей, включающей каждый из модов в модпаке, с полностью переработанной прогрессией боссов с использованием QuestBookUI и tPackBuilder.

Модпак создавался с учётом совместимости с мультиплеером, но мы пока не можем гарантировать его полную работоспособность, так как всё ещё тестируем и разрабатываем его.
